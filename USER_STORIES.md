# Пользовательские истории (веб-помощник)

Коротко: что пользователь хочет сделать и где это в интерфейсе.

| История | Экран / действие |
|--------|-------------------|
| Как пользователь, хочу видеть приветствие и сводку, чтобы понимать объём дел без лишних кликов | **Главная** (`/`) — карточки «Сегодня», «Все задачи», «Рутины», отчёт |
| Как пользователь, хочу быстро добавить задачу текстом или голосом | **Главная**, **Сегодня**, **Действия** — блок «Новая задача» + микрофон / файл |
| Как пользователь, хочу видеть только задачи на сегодня и отметить выполненные | **Сегодня** — чекбоксы, кнопка «Отметить выбранное» |
| Как пользователь, хочу отметить задачи по номерам без поиска по списку | **Сегодня**, **Действия** — «Быстро отметить по номерам» |
| Как пользователь, хочу видеть полный список с глобальными номерами | **Все задачи** |
| Как пользователь, хочу управлять рутинами отдельно | **Рутины** |
| Как пользователь, хочу посмотреть, что уже сделано | **Сделано сегодня**, **За неделю** |
| Как пользователь, хочу удалить, перенести, изменить задачу или отменить выполнение | **Действия** |
| Как пользователь, хочу вспомнить синонимы и правила номеров | **Справка** |
| Как пользователь, хочу выйти из веб-интерфейса | **Выход** в меню |

Навигация: все разделы — в боковом меню; на телефоне — кнопка «☰».
