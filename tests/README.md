# Тесты

Юнит-тесты на логику бота (парсинг дат, времени, извлечение текста задачи).

**Перед доработкой фич** запускай тесты, чтобы не сломать текущее поведение:

```bash
# из корня проекта (c:\Project\bot)
python -m pytest tests/ -v
```

Быстрый прогон без лишнего вывода:

```bash
python -m pytest tests/ -q
```

Остановка на первой ошибке:

```bash
python -m pytest tests/ -x
```

Тесты не требуют базы данных и Telegram; проверяется только модуль `task_parsing`.
